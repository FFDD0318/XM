### 账号注册以及风控
1、使用**不同的手机设备**进行注册、刚注册的账号**不要退出**和切换账号，以免账号被封禁  
2、刚注册的账号**不要立刻跑脚本**，以免账号被封禁，无法提现  
3、同一手机最好不要切换账号，以免cookie过期 (待测)
